When would you want to use a remote repository rather than keeping all your work local?
É possivel trabalhar em conjunto muito mais facilmente de forma remota.

Why might you want to always pull changes manually rather than having Git automatically stay up-to-date with your remote repository?
Pois se o Git fazer atualizações automaticas, pode ser que algum codigo com bug passe para o repositorio no GitHub.

Describe the differences between forks, clones, and branches. When would you use one instead of another?
Fork permite copiar um repositorio direto no GitHub, enquanto clone permite salvar esse diretório no computador do usuário. Branches permitem abortar o projeto de uma outra maneira, de maneira local, sem alterar o arquivo principal(funcional).

What is the benefit of having a copy of the last known state of the remote stored locally?
Facilita alterações no código, permitindo saber qual a versão funcional mais recente.
 
How would you collaborate without using Git or GitHub? What would be easier, and what would be harder?
Fazer algo em grupo sem utilizar o git/github torna tudo mais complicado, dificultando alterações no código, etc.
